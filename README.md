# 世足比分小軟體

手機優先的 FIFA World Cup 2026 比分與數據小工具。

上線網址：https://ggpowermax.github.io/

## 功能

- 今日與下一場賽事
- 完整賽程與階段篩選
- 分組排名
- 射手榜與球隊進球榜
- 單場比賽詳情
- 收藏比賽，使用瀏覽器 localStorage
- 即時 API 失敗時顯示上次成功快取
- 每小時快照資料可覆蓋落後的比分、完賽狀態與技術數據

## 資料來源

使用免金鑰 World Cup 2026 API：

- https://worldcup26.ir/get/games
- https://github.com/rezarahiminia/worldcup2026
